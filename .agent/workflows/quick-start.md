# Быстрый запуск Cellular Life Simulation

## Шаг 1: Запуск локального сервера

// turbo
```bash
cd c:\Users\Дмитрий\.gemini\antigravity\playground\cellular-life
python -m http.server 8080
```

## Шаг 2: Открыть в браузере

Откройте браузер и перейдите на:
```
http://localhost:8080/index.html
```

## Шаг 3: Тестирование

1. **Проверьте начальное состояние:**
   - Синяя ветвистая река должна быть видна
   - Зеленые растения должны появиться рядом с водой

2. **Запустите симуляцию:**
   - Нажмите кнопку "Старт"
   - Откройте консоль браузера (F12)

3. **Подождите 60 секунд и проверьте:**
   - Растения растут вверх
   - Появляются ярко-ЖЕЛТЫЕ семена на верхушках
   - В консоли видны сообщения: `Plant X reached max size`
   - Счетчик семян > 0
